В этом проекте я анализирую датасет с зимними модными товарами за сезоны 2023–2025.
Цель — понять, как отличаются цены, популярность и рейтинги у разных брендов, категорий и статусов трендов, а также проверить данные на ошибки и аномалии.


## Данные

Датасет состоит из **150 строк и 12 столбцов**.
Каждая строка — отдельный товар.

В данных есть:

* бренд, категория, цвет, материал, стиль;
* пол (Men / Women / Unisex);
* сезон;
* цена в долларах;
* популярность;
* рейтинг покупателей;
* статус тренда (Trending, Emerging, Classic, Outdated).


## Первичный анализ

Сначала я загрузил датасет с помощью `pandas`, посмотрел первые и последние строки, размер таблицы и типы данных.

Все столбцы имеют корректные типы:

* числовые — `float` и `int`,
* категориальные — строки.

Дополнительного преобразования типов не потребовалось.


## Обработка данных

### Пропущенные значения

Я проверил данные на пропуски — **пропущенных значений нет**, поэтому заполнять или удалять ничего не пришлось.

### Дубликаты

Проверка на полные дубликаты показала, что **дубликатов нет**.

### Выбросы

Для числовых столбцов (цена, популярность, рейтинг) я проверил выбросы:

* с помощью z-score,
* с помощью IQR.

Критических выбросов не обнаружено — все значения находятся в разумных пределах.


## Статистический анализ

Для числовых данных была посчитана описательная статистика.

Основные результаты:

* средняя цена товара — около **443 USD**;
* минимальная цена — **30 USD**, максимальная — **788 USD**;
* средний рейтинг — **3.8 из 5**;
* средняя популярность — **5.76 из 10**.

Для категориальных признаков я посмотрел количество уникальных значений и их распределение.
Например:

* больше всего товаров у бренда **Mango**;
* самые частые категории — **Scarf, Thermal и Gloves**;
* чаще всего встречается статус **Outdated**, но также много товаров со статусом **Trending**.


## Корреляции

Я построил корреляционную матрицу для числовых признаков.
Сильных зависимостей между ценой, популярностью и рейтингом не обнаружено — корреляции слабые.


## Визуализация

В проекте построено больше 5 графиков:

* гистограмма распределения цен;
* boxplot цен по брендам;
* столбчатая диаграмма количества товаров по категориям;
* линейный график среднего рейтинга по брендам;
* heatmap корреляций;
* boxplot популярности по статусу тренда.

Графики помогают наглядно увидеть различия между брендами, категориями и трендами.


## Группировки и агрегации

Я выполнил несколько группировок:

* по брендам (средняя цена, медиана, рейтинг, популярность);
* по категориям товаров;
* по статусу тренда;
* по сезонам;
* по материалам.

Также была построена сводная таблица (`pivot_table`) со средней ценой товаров по брендам и категориям.
